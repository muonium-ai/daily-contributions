## Goal
Add `os.makedirs(..., exist_ok=True)` calls so scripts don't crash on a fresh clone where `config/` or `data/` directories don't yet exist.

## Acceptance Criteria
- [ ] `find_git_repos.py` creates `config/` before writing `config/repos.txt`
- [ ] `index_loc.py` creates `data/` before connecting to `data/contributions.db`
- [ ] Scripts run successfully from a clean state with no pre-existing directories

## Notes
On a fresh clone, `config/` and `data/` may not exist. Writing to `config/repos.txt` or `data/contributions.db` will raise `FileNotFoundError`.

## Implementation Plan
- [ ] Add `os.makedirs("config", exist_ok=True)` in `find_git_repos.py` before writing `OUTPUT_FILE`
- [ ] Add `os.makedirs("data", exist_ok=True)` in `index_loc.py` before `sqlite3.connect(DB_PATH)`
- [ ] Test by removing `config/` and `data/` directories and running `make all`
