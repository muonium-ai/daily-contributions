## Goal
Move `index_loc.py` logic into `src/daily_contributions/indexer.py` and replace the top-level script with a thin CLI in `tools/index.py`.

## Acceptance Criteria
- [ ] `src/daily_contributions/indexer.py` exposes `parse_numstat`, `run_git_log`, `init_db`, and a `main()` entry point
- [ ] `tools/index.py` is a thin wrapper: `from daily_contributions.indexer import main; main()`
- [ ] `tests/test_numstat.py` imports `parse_numstat` from the new package path
- [ ] `make index` invokes `tools/index.py` and produces the same DB output as before
- [ ] `uv run pytest tests/` is green

## Notes
Keep `init_db`'s migration logic identical (additive `ALTER TABLE` for legacy DBs). The DB path stays relative to the working directory — running `make index` from the repo root must continue to write to `data/contributions.db`.
