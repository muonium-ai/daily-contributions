## Acceptance Criteria
- [ ] `src/daily_contributions/report.py` exposes the print/query helpers (`print_daily_report`, `print_repo_summary`, `print_module_timeline`, `print_tickets_summary`, `print_unarchived_tickets`, `classify_activity`, `get_unarchived_tickets`, `get_non_zero_day_averages`) and a `main()` entry point that calls them in the existing order
- [ ] `tools/report.py` is a thin wrapper: `from daily_contributions.report import main; main()`
- [ ] `tests/test_unarchived_tickets.py` and `tests/test_classify_activity.py` import from the new package path
