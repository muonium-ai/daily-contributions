## Goal
Track the number of unique files touched per day across all repos as a new productivity metric.

## Acceptance Criteria
- [ ] `run_git_log` in `index_loc.py` returns files touched count alongside additions, deletions, commits
- [ ] `daily_loc` table has a `files_touched` column
- [ ] `init_db` handles migration (ALTER TABLE if column missing)
- [ ] `report_generator.py` displays files touched in daily report and averages

## Notes
Use `git log --numstat` output to count unique filenames per day. Each `--numstat` line has `add\tdel\tfilename` — collect unique filenames.
