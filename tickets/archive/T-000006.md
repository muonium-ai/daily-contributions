## Goal
Move the top-level script logic in `find_git_repos.py` into a `main()` function behind an `if __name__ == "__main__"` guard.

## Acceptance Criteria
- [ ] All repo-scanning logic is inside a `main()` function
- [ ] `if __name__ == "__main__": main()` guard is present

- [ ] `uv run find_git_repos.py` still works as before

## Notes
Currently the script runs everything at module scope, making it impossible to import `read_ignore_paths` or `is_ignored` for reuse or testing without triggering the full scan.

## Implementation Plan
- [ ] Move lines 33–50 (the `git_repos`, `ignore_paths`, `os.walk` loop, file write, and prints) into a `def main():` function
